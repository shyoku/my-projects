# 淘宝广告CTR分析

分析阿里天池淘宝广告点击率数据集，使用Python进行多维度分析。

## 数据集
- 来源：[淘宝展示广告点击率预估数据集](https://tianchi.aliyun.com/dataset/dataDetail?dataId=56)
- 包含：用户行为、广告特征、用户画像

## 核心指标
- 总曝光：2655万次
- 总点击率：5.14%
- 分析维度：时间、用户、价格、时段

## 主要发现
1. 周中点击率 > 周末
2. 女性点击率(5.24%) > 男性(4.84%)
3. 中午时段点击率最高(5.18%)
4. <50元商品点击率最高(5.97%)
